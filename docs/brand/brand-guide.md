## Brand feel

**Luxury & premium.** Elegant, minimal, high-end.
Clean white space. Refined typography. Confident use of black and crimson.
The brand communicates exclusivity and trust — the go-to real estate authority in Suriname.

## Color palette

| Name | Hex | RGB | Usage |
|---|---|---|---|
| **Crimson** | `#8B0000` | 139, 0, 0 | Primary brand color. CTAs, accents, highlights, active states. |
| **Crimson Dark** | `#6B0000` | 107, 0, 0 | Hover state for crimson elements. |
| **Crimson Light** | `#A50000` | 165, 0, 0 | Subtle accent variant. |

| **Light grey** | `#E8E4E0` | 232, 228, 224 | Borders, dividers, subtle backgrounds. |
| **Medium grey** | `#9E9A96` | 158, 154, 150 | Secondary text, placeholders, captions. |

### Usage rules
- **Never** place crimson text on a red background.
- **Never** use pure `#FF0000` — always use `#8B0000` (Crimson).
- Off-white (`#F8F6F3`) is the default page background — not white, not grey.
- Black is for text and structural elements. Crimson is for action and emphasis only.
- White space is a design element — use it generously.

## Typography

### Fonts
| Role | Font | Weight | Source |
|---|---|---|---|
| Display / Hero headlines | **Cormorant Garamond** | 300, 400, 600 | Google Fonts |
| Section headings (H2–H3) | **Cormorant Garamond** | 400, 600 | Google Fonts |
| Body text, UI, labels | **Inter** | 300, 400, 500, 600 | Google Fonts |
| Price display | **Cormorant Garamond** | 300 | Google Fonts |

### Type scale
| Element | Font | Size | Weight | Line height |
|---|---|---|---|---|
| Hero headline | Cormorant Garamond | 4.5rem (72px) | 300 | 1.1 |
| H1 | Cormorant Garamond | 3rem (48px) | 400 | 1.15 |
| H2 | Cormorant Garamond | 2.25rem (36px) | 400 | 1.2 |
| H3 | Cormorant Garamond | 1.5rem (24px) | 600 | 1.25 |
| Body large | Inter | 1.125rem (18px) | 400 | 1.7 |
| Body | Inter | 1rem (16px) | 400 | 1.7 |
| Small / caption | Inter | 0.875rem (14px) | 400 | 1.5 |
| Label / tag | Inter | 0.75rem (12px) | 600 | 1.4 |
| Price | Cormorant Garamond | 2rem (32px) | 300 | 1.2 |
| Navigation | Inter | 0.9375rem (15px) | 500 | — |

### Typography rules
- Headlines use Cormorant Garamond with generous letter-spacing (`tracking-wide`).
- Never use Cormorant Garamond below 18px — it loses elegance at small sizes.
- Body text is always Inter. Never use Cormorant for long-form reading.
- Prices are always Cormorant Garamond — this is a deliberate luxury signal.
- All caps should only be used for labels/tags in Inter, never in Cormorant.

## Spacing & layout

- Base spacing unit: `8px`.
- Generous padding — minimum `24px` on mobile, `48px+ ` on desktop sections.
- Max content width: `1280px` centered.
- Listing card grid: `3 columns` desktop, `2 tablet`, `1 mobile`.
- Use whitespace as a luxury signal — never crowd elements.

## Component visual identity

### Buttons
- **Primary CTA:** Crimson background (`#8B0000`), white Inter text, no border-radius or subtle `2px` radius only. Hover: Crimson Dark (`#6B0000`).
- **Secondary:** Black background, white text. Hover: Charcoal.
- **Ghost:** Transparent with `1px` crimson border, crimson text. Hover: fill crimson.
- No rounded pill buttons — sharp or very slightly rounded only (`border-radius: 2px`).

### Cards (listing cards)
- White background, `1px` light grey border.
- Subtle shadow: `0 2px 12px rgba(0,0,0,0.06)`.
- Hover: shadow deepens, slight `translateY(-2px)`.
- Status badge: crimson background for `new`, black for `sold`/`rented`, charcoal for others.
- Price in Cormorant Garamond, address in Inter.

### Navigation
- Black background, white Inter text.
- Crimson underline or dot on active/hover links.
- Logo left, nav links right, CTA button (crimson) far right.

### Footer
- Black background.
- White Inter text.
- Crimson for section headings and hover links.

## Imagery

- Architectural photography: clean, well-lit, professional.
- Avoid stock photo feel — prefer real property photography.
- Hero images: high-contrast, dramatic framing.
- No filters or heavy color grading — let the color palette speak.

---

## What is NOT the brand

- No gradients.
- No drop shadows heavier than `0 4px 24px rgba(0,0,0,0.12)`.
- No rounded corners beyond `4px`.
- No playful or friendly fonts.
- No orange, purple, green, or any color outside the palette.
- No busy patterns or textures — flat surfaces only.
